# 옵티마이저 

사용자가 질의한 sql문에 대해 최적의 실행 방법을 결정하는 역할을 수행한다. 
이러한 최적의 실행 방법을 실행계획 이라고 한다. 
관계형 데이터베이스는 궁극적으로 sql문을 통해서만 데이터를 처리할 수 있다. 
JAVA, C 등과 같은 프로그램 언어와는 달리 SQL 은 사용자의 요구사항만 기술할 뿐 처리과정에 대한 기술은 하지않는다.
