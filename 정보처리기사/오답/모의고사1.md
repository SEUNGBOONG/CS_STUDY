
# 레지스터 

- CPU내에서 사용되는 고속의 소형 메모리
- 레지스터는 데이터를 임시로 저장 - > 명령어 실행 연산을 위해 빠르게 접근할 수 있도록 설계
- 고속 접근시간
- 작은 저장용량
- 임시저장 
- 빠른 데이터 접근
- 임시데이터저장
- 명령어 실행
