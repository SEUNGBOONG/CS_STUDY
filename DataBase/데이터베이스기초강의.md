2. CREATE TABLE MYUSER

- VARCHAR(30): 30만큼 하나의 데이터를 저장할때 30Byte 사용 한다.
- VARCHAR2 : 크기가 들어가는 데이터만큼 자동조절되는 VARCHAR2
- PRIMARY KEY: 고유키, 각각 다른 값을 가진다. 테이블의 대표 기준이되는 데이터이다. 기본키 or PRIMARY KEY라고부른다.
- 기본키를 하나 이상 하나 쯤은 설정을 한다.
